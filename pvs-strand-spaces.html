Strand Spaces in PVS

John D. Ramsdell
The MITRE Corporation
January 2015

This package contains several libraries of PVS theories for reasoning about strand spaces, state, and security goals. The theories are covered by a BSD-style license.

This package offers two formulations of strand spaces. All of the original work assumes the message algebra is an initial algebra. This assumption simplifies the theory of the message algebra because there is no need to define and consider matching and substitution. Our work on protocols and state makes use of this simplifying assumption.

In an effort to support future work on protocol transformations and security goals, the new formulation of strand spaces makes use of a message algebra that is a free algebra generated by an infinite set of message variables. It specifies a matching algorithm and verifies its correctness.

Strand spaces over a free message algebra with variables
skeletons
The foundational definitions. Defines strand spaces and bundles. Although possible, skeletons have not yet been defined.
security_goals
An example of using the Node-Oriented Shape Analysis Sentences in Appendix B within this formalism.
Strand spaces over an initial message algebra
strand_spaces
The foundational definitions. Defines strand spaces and bundles over an initial message algebra.
blanchet
An example of using Shape Analysis Sentences within this formalism.
state_spaces
Defines the notion of compatibility between bundles and state transitions.
stap and envelope
Show how to reason about protocol executions in the presence of the long term state provided by a Trusted Platform Module. The stap library contains a proof of a security goal for the Simple TPM Attestation Protocol and the envelope library has a more involved proof of a security goal for the Envelope Protocol.
award_card
A very simple example of state as boxes being checked on an award card.
diffie_hellman
An algebra with an Abelian group for exponents.
neutral_nodes
Strands spaces augmented with labels as state synchronization events.
cpsa
Strands spaces with state as implemented in an experimental version of CSPA.
guarded_transitions
Strands spaces augmented with guarded transitions as state synchronization events.
wrap_decrypt
An analysis of the Wrap-Decrypt protocol using guarded transitions.
log
Contains a script that validates the proofs in the various theory libraries.
